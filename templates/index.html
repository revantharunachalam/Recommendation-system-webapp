Movie Recommendation System

  • Instructions for using API
  • 1. Use 'URI/recommend?name=movie_name' to get the recommended movies
  • 2. Use 'URI/sentiment?name=movie_name' to get the sentiment analysis of movies
  • 3. For space use '+' or '%20'
  • 4. 'URI/home' or 'URI/' to see the instructions
  • 5.The response from the API consist of movie name, Director, Genere, Actors of recommended movies followed by the same details of query movie
